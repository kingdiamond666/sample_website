Important HTML Tags

This page is designed to be a quick reference for some of the common tags covered in Learn Enough HTML to be Dangerous.In the process we'll learn how to make HTML tables via table and related tags.

The tables don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings including a heading (level 1–6)
p paragraph include a paragraph
table table including a table
tr table row including a row of data
th tabe header mark a table header
td table data including a table data cell
header header element creates a header element
span span element creates a span element
div div element creates a div element
ol ordered list creates an ordered list
ul unordered list creates an unordered list
li list creates a list item
blockquote block quotation show formatted quotation
br break enter line break

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="http://learnenough.com/">Learn Enough</a> Learn Enough
img image add an image <img src="images/picture.jpg" alt="picture.jpg"> picture.jpg
code code format as source code <code>table</code> table
span span define incline section in doc <span>Call me Ishmael</span> Call me Ishmael

Document Tags

Tag Name Purpose
html HTML defines skeleton as html
title TITLE defines page title
head Head defines info about page
body Body the actual content of a page
meta Meta data about the data
